문제 {
    하나의 수열에는 다양한 수가 존재하며, 이런 큰 수는 크기와 상관 없이 무작위로 주어진다. 

    수열을 내림차순으로 정렬하는 프로그램을 만드시오.
}
입력 조건
{
    첫째 줄에 수열에 속해 있는 수의 개수 N이 주어진다. 이때 범위는 1 <= N <= 500

    둘째 줄부터 N + 1 번째 줄 까지 N개의 수가 입력된다. 수의 범위는 1 이상 100,000 이하 자연수
}
 
출력 조건
{
   입력으로 주어진 수열이 내림차순으로 정렬된 결과를 공백으로 구분해서 출력하면된다. 
}
 

입력 예시
{
    3
    15
    27
    12
}
 

출력 예시
{
    27 25 12
}


해결 방법
{
    기본적 아이디어 : 정렬 라이브러리 사용하기(내림 차순으로 출력)

    실제 해결 방법
    input 값 입력받기(리스트)
    리스트.sort(rev = True)
    print(리스트)
}

보안할 점
{
    출력 예시와 동일하게 출력하는 것이 가장 중요하다.

    값을 입력받는 과정과 활용하는 과정을 좀 더 익숙해질 필요가 있다.
}
