문제 
{
    가로의 길이가 N, 세로의 길이가 2인 직사각형 형태의 얇은 바닥이 있다. 태일이는 이 얇은 바닥을 1 X 2의 덮개, 

    2 X 1의 덮개, 2 X 2의 덮개를 이용해 채우고자 한다.

    이때 바닥을 채우는 모든 경우의 수를 구하는 프로그램을 작성하시오. 

    예를 들어 2 X 3 크기의 바닥을 채우는 경우의 수는 5가지이다.
}

입력 조건
{
    1. 첫째 줄에 N이 주어진다. (1 <= N <= 1,000)
}
 
출력 조건
{
    첫째 줄에 2 X N 크기의 바닥을 채우는 방법의 수를 796,796으로 나눈 나머지를 출력한다.
}
 

입력 예시
{
    3
}
 

출력 예시
{
    5
}


해결 방법
{
   점화식을 사용하여 문제를 해결한다.

   N=1 : 1
   N=2 : 1+1+1
   N=3 : 1+1+1+1+1

   점화식이 잘 보이지 않지만, 위의 보기에서 나온 것과 같이 총 3가지 종류의 타일이 존재하며

   각 타일이 들어갈 수 있는 경우와 없는 경우를 나누어서 생각하면 생각하기 쉬어진다. 우선 가장 기본이 되는 타일인 1x2 타일의 경우

   자리만 존재한다면 언제든지 들어갈 수 있다. 나머지 두 타일의 경우 가로 공간이 2이상이 남아야만 들어갈 수 있다.

   따라서, i번쨰 타일을 집어 넣을 때 공간이 하나가 남은 경우와 2개가 남은 경우를 나누어서 더해주면 해결 가능하다.
   
   i번쨰 상황에서 공간이 하나만 남았을 경우에는 이전의 값과 동일한 값을 넣어주면 된다. answer[i] = answer[i-1]

   만약 2개의 공간이 남은경우에는 answer[i] = answer[i-2]로 진행해주면된다. 공간이 2개 남은 경우에는 타일의 종류도 2종류이기 때문에 2를 곱해주면 된다.

   따라서, 점화식은 answer[i] = answer[i-1] + 2*answer[i-2]
}
